源自 ExoPlayer library 的 Android 影片播放器。使用 ExoPlayer FFmpeg 元件並支援所有音訊格式 (甚至能夠對應如 AC3、EAC3、DTS、DTS HD、TrueHD 等特殊格式)。

使用藍牙耳機 / 揚聲器時，能夠準確同步音訊及視訊軌道。

支援格式

 * 音訊：Vorbis、Opus、FLAC、ALAC、PCM / WAVE (μ-law、A-law)、MP1、MP2、MP3、AMR (NB、WB)、AAC (LC、ELD、HE；Android 9+ 的 xHE)、AC-3、E-AC-3、DTS、DTS-HD、TrueHD
 * 視訊：H.263、H.264 AVC (基礎配置；Android 6+ 主要配置)、H.265 HEVC、MPEG-4 SP、VP8、VP9、AV1
 * 封裝：MP4、MOV、WebM、MKV、Ogg、MPEG-TS、MPEG-PS、FLV
 * 串流：DASH、HLS、SmoothStreaming、RTSP
 * 字幕：SRT、SSA、ASS、TTML、VTT

在相容 / 支援的裝置上播放 HDR (HDR 10+ 和 Dolby Vision)。

特色

 * 音訊 / 字幕軌道切換
 * 播放速度控制
 * 左右滑動及輕按兩次以快速跳轉
 * 上下滑動以調整螢幕亮度 (左側) / 音量 (右側)
 * 雙指縮放 (Android 7+)
 * Android 8+ 子母畫面 (Android 11+ 將可調整大小)
 * 顯示模式 (符合視窗 / 裁剪畫面)
 * 音量增強
 * 為 Android 電視盒自動偵測影格速率 (Android 6+)
 * 播放後動作 (刪除檔案 / 下個影片)
 * 觸控鎖定 (長按)
 * 沒有廣告、追蹤和過多權限

如要載入外掛 (非嵌入式) 字幕，請按住畫面底部中的「資料夾」圖示。首次操作時，系統將會引導您設定主要影片資料夾以啟用自動載入字幕。
